CSS Combinators

                Combinators combine the selectors to provide them a useful relationship
                 and the position of content in the document. This comes in handy when 
                 you want to style certain elements all at once rather than styling elements individually.
                 CSS combinator selectors are as follows.
                  1. Descendant Selector (space)
                  2. Child Selector (>)
                  3. Adjacent Sibling Selector (+)
                  4. General Sibling Selector (~)
